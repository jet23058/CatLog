隱私權政策

生效日期：2026年1月24日

歡迎使用「極簡貓資產」（以下簡稱本應用程式）。我們非常重視您的隱私權，本隱私權政策將說明我們如何收集、使用及保護您的個人資訊。

1. 資訊收集與使用

本應用程式使用 Google Firebase Authentication 進行身份驗證。當您登入時，我們會收集您的 Google 帳戶基本資訊（如電子郵件地址、顯示名稱及個人頭像），僅用於識別您的身份並同步您的資產資料。

您的資產紀錄、收支數據等財務資訊均加密存儲於 Google Cloud Firestore 資料庫中，僅供您個人存取與檢視。

2. 生物辨識資料

若您啟用 Face ID / Touch ID 鎖定功能，本應用程式將使用 Web Authentication API 與您的裝置進行驗證。所有的生物辨識特徵（如指紋或臉部資訊）皆安全地存儲於您的裝置硬體中，本應用程式無法獲取、存儲或傳輸這些原始生物辨識資料。

3. 資料安全

我們採用業界標準的加密技術來保護您的數據傳輸與存儲安全。然而，請注意沒有任何網路傳輸或電子存儲方式是 100% 安全的。

4. 第三方服務

本應用程式可能會使用以下第三方服務：

  • Google Firebase：用於身份驗證、資料庫存儲與託管。
  • Dropbox（可選）：若您使用 Dropbox 匯入功能，我們將請求存取您指定的 CSV 檔案，但不會存取您 Dropbox 中的其他檔案。

5. 變更通知

我們可能會不時更新本隱私權政策。我們建議您定期查看本頁面以獲取任何變更。變更後的政策一經發布即生效。

6. 聯絡我們

如果您對本隱私權政策有任何疑問，請透過 GitHub Issues 或應用程式內的聯絡方式與開發者聯繫。
